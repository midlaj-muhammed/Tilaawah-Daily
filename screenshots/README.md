# Screenshots Directory

This directory contains screenshots for the Tilaawah Daily app README.md file.

## File Naming Convention

Use descriptive names following this pattern:
- `home-screen.png` - Main dashboard
- `daily-verse.png` - Daily verse display
- `progress-tracking.png` - Progress charts
- `settings.png` - Settings screen
- `notifications.png` - Notification settings
- `profile.png` - User profile screen

## Image Specifications

- **Format**: PNG (recommended) or JPG
- **Dimensions**: 
  - Mobile: 375x812px (iPhone X/11/12)
  - Tablet: 768x1024px (iPad)
- **Quality**: High resolution, minimum 72 DPI
- **Size**: Keep under 500KB per image for fast loading

## Tips for Good Screenshots

1. Use a clean device frame or no frame
2. Show the app in both light and dark modes if supported
3. Ensure text is readable and UI elements are clear
4. Avoid showing sensitive user information
5. Use consistent styling across all screenshots
6. Consider using device mockups for better presentation

## Tools for Creating Screenshots

- **iOS**: Use Xcode simulator + Cmd+S
- **Android**: Use Android Studio emulator + Cmd/Ctrl+S
- **Device Frames**: Use tools like Frame.js or DeviceFrames
- **Online Tools**: Canva, Figma, or dedicated screenshot tools
